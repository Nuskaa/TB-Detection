How It Works

01

Upload X-Ray

Choose a chest X-ray JPG or PNG. Drag-and-drop is supported.

02

Preprocessing

Grayscale conversion → resize to 64×64 → flatten to 4,096 features → StandardScaler normalisation.

03

Dual Inference

SVM (RBF kernel) and Logistic Regression both classify the feature vector independently.

04

Full Report

Severity scoring, edge overlay, preprocessing pipeline view, symptom checker, and a downloadable PDF.
